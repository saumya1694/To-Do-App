📝 To Do App

A modern To Do web application built using HTML, CSS, and JavaScript that helps users manage daily tasks efficiently. The app provides a clean and interactive interface where users can add tasks, mark them as completed, and delete them when finished.

Tasks are stored using localStorage, ensuring that the task list remains saved even after refreshing or reopening the browser.

🚀 Features

* Add new tasks quickly
* Mark tasks as completed
* Delete tasks easily
* Tasks persist using localStorage
* Modern responsive UI design
* Simple and user-friendly interface

🛠 Technologies Used

* HTML
* CSS
* JavaScript

🎯 How to Use

1. Enter a task in the input field
2. Click Add or press Enter
3. Click on a task to mark it as completed
4. Use the delete button to remove a task
